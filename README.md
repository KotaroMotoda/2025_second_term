## 用語解説

### マルチスレッド
複数のスレッド（実行の流れ）を切り替えたり同時に実行したりできる仕組み全般を指す。ソフトウェア（OSやランタイムによるスレッド切替）とハードウェア（CPUによる同時/高速切替）の両方を含む。

ハードウェア的マルチスレッドには以下が含まれる：
- **SMT（同時マルチスレッディング）**
- **粗粒度マルチスレッド**
- **細粒度マルチスレッド**
